# Homework 6
## Generative art
### Emerge

本次作业主要制作了自动生成的艺术画面，模仿了电流和水流的运动形态  
用户能够通过键盘输入，并且保存视频mp4。  
具体操作键如下：  
``s``开始录制  
``q``停止录制  

记住以上操作就可以试一试啦～
